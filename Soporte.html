ENCONTRÁ LA INFORMACIÓN QUE NECESITÁS PARA LA INSTALACIÓN Y USO DEL PRODUCTO

1. Instalación: Asegúrese de contar con un sistema compatible, conexión a Internet y herramientas como pip o npm para instalar librerías. Verifique que el software funcione correctamente antes de usarlo.
2. Pantalla Principal: Registrar: Captura datos e imagen facial de los estudiantes para futuros reconocimientos. Ingresar: Utiliza reconocimiento facial para registrar la entrada del estudiante.
3. ESP32: Conecta el software con dispositivos físicos (como LEDs). Configure Wi-Fi y pruebe la conexión con comandos simples.
4. Indicadores LED: Verde: Acceso autorizado. Rojo: Acceso denegado, el estudiante no está registrado.
5. Wi-Fi: Una conexión estable es esencial para sincronizar con la base de datos y servicios.
6. Administración: Gestione, edite, elimine o visualice datos de estudiantes desde la interfaz.
7. Recomendaciones: Mantenga el software y los dispositivos actualizados; consulte soporte técnico en caso de problemas.
